# Next.js Learning Journey 🚀

このリポジトリは、Next.jsを基礎から実践まで学び、使いこなすことを目標とした学習の記録です。
作成したアプリケーションや学習内容をまとめています。

## 🛠️ 使用技術 (Tech Stack)

- **Framework**: Next.js, React
- **Language**: TypeScript
- **Authentication**: Supabase
- **Styling**: CSS Modules, (Tailwind CSS - setup)
- **Lint**: ESLint
